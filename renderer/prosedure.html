Prosedur Pemakaian

Persyaratan Sebelum Menggunakan

  • Pengguna adalah pihak yang berwenang dan memiliki akses ke website terkait.
  • Pastikan perangkat terhubung ke koneksi internet yang stabil.

Memulai Automatisasi

  • Klik tombol GO untuk memulai proses automatisasi.
  • Setelah automatisasi berjalan, jangan lakukan interaksi apapun dengan halaman target (klik, pindah, refresh, scroll).
  • Jika perlu mengerjakan tugas lain, duplicate tab browser. Tab duplicate tidak akan terpengaruh oleh program.

Navigasi Tab

  • Tab Logs: Menampilkan langkah-langkah proses dan aktivitas program secara real-time.
  • Tab Data: Menampilkan hasil data yang berhasil dikumpulkan.
  • Gunakan tombol Export to CSV di tab Data untuk menyimpan data ke dalam file .csv.

Penyimpanan & Ekspor Data

  • Data hanya tersimpan selama aplikasi aktif dan akan hilang jika aplikasi ditutup.
  • Menutup aplikasi akan menghapus seluruh data dan menghentikan automatisasi.
  • Disarankan untuk rutin mengekspor data demi menghindari kehilangan data jika terjadi crash atau force close.
